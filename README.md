# cifar10_CNN
Python Script to identify images of CIFAR-10 dataset using Keras!

## Getting Started

### Prerequisites

    Python 3.6, Keras 2.1.6
    
### Dataset

 The dataset can be downloaded [here](https://www.cs.toronto.edu/~kriz/cifar.html)
